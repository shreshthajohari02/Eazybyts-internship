# Event-management-app-
Milestone is a modern event management mobile app UI design created in Figma. The app allows users to discover events, RSVP, book tickets, and make payments easily. It includes screens for login, event search, event details, notifications, booking, and profile management, focusing on a clean and user-friendly experience.
# Digital-Marketing-website
UI/UX design project for a Digital Marketing Agency website created in Figma. Includes multi-page layouts with responsive designs for desktop and mobile. Focused on user flow and prototyping.
